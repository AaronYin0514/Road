# 第 2 章:jQuery 中的 AJAX

## 2.1 get 请求

```javascript
$.get(url, [data], [callback], [type])
```

* url:请求的 URL 地址。
* data:请求携带的参数。
* callback:载入成功时回调函数。 
* type:设置返回内容格式，xml, html, script, json, text, _default。

## 2.2 post 请求

```javascript
$.post(url, [data], [callback], [type])
```

* data:请求携带的参数。
* callback:载入成功时回调函数。
